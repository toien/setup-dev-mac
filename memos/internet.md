### generic proxy

for programs like `curl`, `brew`, `gradle`, `android sdk`, `flutter sdk`, `npm`, `pyenv`,
they will obey env vars like:

```
HTTP_PROXY
http_proxy
HTTPS_PROXY
ALL_PROXY
NO_PROXY
```

set/unset these variables by functions defined in [~/.zshrc](../assets/.zshrc)
